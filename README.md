# 表格整理小程序

一个纯前端小工具，支持：

- 上传 Excel/CSV
- 根据表头自动识别字段类型
- 按任意列排序（升序/降序）
- 按“分类列”做分类筛选
- 按每个字段做关键字筛选（包含匹配）

## 使用方式

1. 直接双击打开 `index.html`，或在当前目录启动静态服务。
2. 上传 `.xlsx` / `.xls` / `.csv` 文件。
3. 选择排序列、排序方向、分类列和分类值，点击“应用排序/筛选”。

## 说明

- 默认会自动尝试把包含“分类/类别/type/status”等字段当作分类列。
- 字段自动分类是基于常见关键词规则，后续可按你的业务再扩展。
